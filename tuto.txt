Description du programme :

Bonjour et merci d'avoir choisi TextGameEngine pour vous évader le temps d'un instant.
Ce programme est une interface graphique permettant de créer ses propres histoires à l'aide d'une
interface simple, permettant d'associer monstres eet items dans une histoire passionante.
Des centaines de possibilités sont réalisables et vous pouvez vous arrêter en pleine conception
pour la reprendre plus tard.
Une histoire est une suite de questions dont les réponses vous emmènent vers d'autres questions.
L'Engine vous permet donc de d'inventer vos propres questions, en créant la description d'une situation,
et deux réponses, en créant les choix et en les liant à la situation suivante.


Comment créer une histoire :

Pour créer une nouvelle histoire, rien de plus simple. Il vous suffit de cliquer sur le bouton dédié.
Vous allez ensuite arriver sur une page d'édition.
Vous pouvez ensuite rentrer les informations de chaq étape dans les cases faites pour cela. N'oubliez
pas de valider une fois que tout est terminé !

Vous pouvez aussi modifier votre histoire déjà créée au moment où vous le souhaitez en appuyant
sur le bouton "modifier une histoire"


Comment jouer à un de vos jeux :

Afin de jouer, il vous suffit de cliquer sur le bouton "jouer à un jeu". L'explorateur de fichier s'ouvre
pour que vous puissiez choisir un des fichier .CSV associés à une histoire. Ensuite retournez sur la
fenêtre de jeu qui sera apparue. Votre histoire est chargée ! Faites les bons choix et amusez-vous
à découvrir toutes les fins possibles !
